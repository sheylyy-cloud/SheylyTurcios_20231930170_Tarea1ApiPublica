## 🎯 Descripción
Este proyecto es una aplicación de consola en C# que consume una API pública.

Se utilizó la API de Rick and Morty para obtener información de los personajes de la serie.

API utilizada:
https://rickandmortyapi.com/

## 🚀 Funcionalidades

El programa permite:

1. Mostrar un listado de personajes.
2. Buscar un personaje específico por ID.
3. Mostrar en consola:
   - Nombre
   - Especie
   - Estado

## 🛠 Tecnologías Utilizadas

- C#
- .NET
- HttpClient
- Newtonsoft.Json
- Visual Studio 2022

## 🔌 Ejemplo de uso

Al ejecutar el programa se muestra el siguiente menú:

=== API Rick and Morty ===
1. Mostrar lista de personajes  
2. Buscar personaje por ID  

Si el usuario elige la opción 2 e ingresa el ID 1, el resultado será:

Nombre: Rick Sanchez  
Especie: Human  
Estado: Alive
